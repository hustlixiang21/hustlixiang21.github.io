什么是git？

Git 是一个开源的分布式版本控制系统，用于敏捷高效地处理项目。

版本控制

能够轻易的将仓库里的东西恢复到上一次的commit。

原理：采用快照的方式进行版本控制。

分布式

我的理解就是能够多人多台主机在不同时空进行同一个开发，且互不影响，最后统一提交。

基本原理

分为工作区、暂存区与版本库，文件夹中有一个.git文件，里面是暂存区和版本库，而.git外的区域就是工作区。
